Memory

This poster was made to appropriate a meaningful memory into a visual medium. This particular memory was about a trip I made to my home country of Vietnam for the first time. This was a life-changing trip and experience.

I wanted to showcase the different aspects that make up the country. Each imagery was illustrated to show that Vietnam has changed but still kept its core identity and values. I wanted to show that through the work, as it was very important to me. Another key aspect of it was how important trip was to me, as it was my very first time visiting Vietnam. That is why I chose this memory in the first place.
